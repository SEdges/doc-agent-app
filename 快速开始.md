# Joern Docs Agent - 快速开始指南 🚀

这是一个简化的中文安装和使用指南。完整文档请参阅 [README.md](README.md)。

---

## 📋 准备工作

### 你需要准备：

1. ✅ **Python 3.10+** - [下载地址](https://www.python.org/downloads/)
2. ✅ **Conda** - [Anaconda](https://www.anaconda.com/download) 或 [Miniconda](https://docs.conda.io/en/latest/miniconda.html)
3. ✅ **OpenAI API Key** - 从 [OpenAI Platform](https://platform.openai.com/api-keys) 获取

### 检查环境

```powershell
# 检查 Python 版本
python --version  # 应该显示 3.10 或更高

# 检查 Conda
conda --version
```

---

## 🛠️ 安装步骤（5 分钟）

### 1️⃣ 创建 Python 环境

```powershell
# 创建名为 joern-docs 的环境
conda create -n joern-docs python=3.10 -y

# 激活环境
conda activate joern-docs
```

### 2️⃣ 安装依赖包

```powershell
# 确保在项目目录中
cd C:\Users\你的用户名\doc-agent-app

# 安装所有依赖
pip install -r requirements.txt
```

### 3️⃣ 配置 API Key

```powershell
# 复制启动脚本
Copy-Item start_app.example.ps1 start_app.ps1

# 用记事本打开
notepad start_app.ps1
```

在打开的文件中，找到**第 20 行**，替换为你的 API Key：

```powershell
# 修改前
$env:OPENAI_API_KEY = "sk-proj-YOUR_API_KEY_HERE"

# 修改后（填入你的真实 Key）
$env:OPENAI_API_KEY = "sk-proj-eQEwQPKeSwHbV89g..."
```

保存并关闭文件。

### 4️⃣ 启动应用

```powershell
# 如果遇到执行策略错误，先运行这个
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

# 启动应用
.\start_app.ps1
```

你应该会看到：

```
[START] Launching Joern Docs Agent...
[OK] API Key set (last 4 chars: ...DwsA)
[ENV] Activating conda environment: joern-docs
[DIR] Working directory: C:\Users\...
[RUN] Starting Streamlit app...

  You can now view your Streamlit app in your browser.
  Local URL: http://localhost:8501
```

### 5️⃣ 构建知识库（第一次使用）

1. 浏览器会自动打开 `http://localhost:8501`
2. 点击左侧 **"🚀 开始抓取并构建"** 按钮
3. 等待 3-5 分钟（会抓取所有文档并建立索引）
4. 看到 "🎉 知识库构建完成！" 后即可开始提问

### 示例对话 1：基础了解

```
你: Joern 是什么？

AI: Joern 是一个代码分析平台，用于分析源代码、字节码和二进制代码。
    它生成代码属性图（Code Property Graph, CPG）...

你: 它支持哪些语言？

AI: （AI 理解"它"指的是 Joern）
    Joern 支持以下编程语言：
    1. C/C++ - 成熟度：非常高
    2. Java - 成熟度：非常高
    ...
```

### 示例对话 2：深入探讨

```
你: Code Property Graph 是什么？

AI: CPG 是一种将代码的多个抽象层次整合到一个统一数据结构中的图表示...

你: 它包含哪些节点类型？

AI: CPG 主要包含以下节点类型：
    1. AST 节点（抽象语法树）
    2. CFG 节点（控制流图）
    ...

你: 这些节点之间有什么关系？

AI: （继续基于上下文）节点之间通过边连接，主要有三类边：
    1. AST 边 - 表示语法结构
    2. CFG 边 - 表示控制流
    ...
```

## ❓ 常见问题速查

### 问题 1：脚本无法运行

**错误提示**：
```
无法加载文件 xxx.ps1，因为在此系统上禁止运行脚本
```

**解决办法**：
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\start_app.ps1
```

---

### 问题 2：API Key 错误

**错误提示**：
```
AuthenticationError: Error code: 401
```

**检查清单**：
1. ✅ 是否正确填写了 API Key（第 20 行）
2. ✅ API Key 是否有效（登录 OpenAI 平台查看）
3. ✅ 账户是否有余额

---

### 问题 3：文档不完整

**现象**：AI 只能回答快速入门相关的问题

**解决办法**：

```powershell
# 1. 停止应用（Ctrl+C）

# 2. 删除旧数据库
Remove-Item -Recurse -Force .\joern_docs_db\

# 3. 重新启动
.\start_app.ps1

# 4. 重新构建知识库（点击"开始抓取并构建"）
```

构建完成后，应该看到 **抓取了 30-100 个页面**。

---

### 问题 4：端口被占用

**错误提示**：
```
Port 8501 is already in use
```

**解决办法**：

```powershell
# 方法 1：关闭占用端口的程序
netstat -ano | findstr :8501
taskkill /PID <进程ID> /F

# 方法 2：使用其他端口
streamlit run joern_doc_agent_app.py --server.port 8502
```
